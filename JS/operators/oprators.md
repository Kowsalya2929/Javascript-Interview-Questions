**Arithmetic Operator:**

1. **Addition(+):**

```js
let x = 10;
let y = 20;
let z = x + y;
console.log(z) //30
```
2. **Subtraction(-):**

```js
let x = 10;
let y = 20;
let z = x - y;
console.log(z) //10
```

3. **Multiplication(*):**

```js
let x = 10;
let y = 20;
let z = x * y;
console.log(z) //200
```

4. **Division(/):**

```js
let x = 40;
let y = 20;
let z = x / y;
console.log(z) //2
```

5. **Modulus(Remainder)(%):**

```js
let x = 20;
let y = 10;
let z = x % y; 
console.log(z) //0
```

6. **Exponentiation:**

```js
let x = 2;
let z = x ** 2;
console.log(z) //4
```

7. **Increment(++):**

```js
//Post Increment

let x = 2;
console.log(x++) //2
console.log(x)   //3

//Pre Increment

let y = 2;
console.log(++y) //3
console.log(y)   //3
```

8. **Decrement Operator(--):**

```js
//Post Decrement

let x = 2;
console.log(x--) //2
console.log(x)   //1

//Pre Decrement

let y = 2;
console.log(--y) //1
console.log(y)   //1
```
